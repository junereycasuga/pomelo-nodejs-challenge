# Pomelo NodeJS Challenge

A small program that maps children of its parent

### Running the project

1. Clone the project

```sh
git clone git@github.com:junereycasuga/pomelo-nodejs-challenge.git
```

2. Install dependencies

```sh
npm i
# or npm install
```

3. Setup environment variables

```sh
cp .env.example .env
```

4. Run it!

```sh
npm run dev
```

### Running the test

```
npm run test
```
